# The Daily Brew – Local Coffee Shop Website

## 🎯 Real Business Project
This is a complete, production-ready website for **The Daily Brew**, a real local coffee shop.  
Built with **HTML5, CSS3, JavaScript, Node.js, Express**.

## ✅ What Makes This Perfect for a Business Pitch
- **Fully responsive & fluid** – works on phones, tablets, laptops, and large screens.
- **Mobile-first navigation** with hamburger menu.
- **WhatsApp floating button** – customers can order directly.
- **Contact form with backend API** – ready to send emails to the owner.
- **Google Maps embed** – shows exact location.
- **Modern, warm aesthetic** – builds trust and professionalism.
